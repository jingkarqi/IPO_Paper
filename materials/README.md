# Materials

`materials/` 只放研究方法资产和写作资产，不放脚本输出。

## 子目录

- `core/`: 当前主流水线直接使用或直接依赖的材料。
- `reference/`: 仍有价值、但不作为当前主口径的补充说明。
- `drafts/`: 提纲、摘要、引言等写作草稿。
- `planning/`: 进度和安排。

## 当前主口径文件

- `core/ai_keywords_weighted.csv`
- `core/annotation_manual.md`
- `core/data_collection_checklist.md`
- `core/llm_prompt_template.md`
- `core/sample_screening_rules.md`
- `core/variable_definition_table.csv`

`reference/ai_keyword_dictionary_basic.csv` 是较简化的备用关键词表，可在补充筛词或人工扩词时参考。
